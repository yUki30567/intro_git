Gitの練習をしていきます。
基本的なワークフローを学びます。

対象ファイルが格納されているフォルダへcdで移動
git statusでファイルの変更状態を確認

ステージングエリア(git add)　→ ローカルリポジトリ(commit)　→ GitHubリポジトリ(push)

・ステージングエリア　※控室
・ローカルリポジトリ　※端末内のGitリポジトリ
・グローバルリポジトリ　※GitHubリポジトリ

「git commit -v」で変更内容を確認しながらコミットできる
この時にコミットメッセージをVIMエディターで記載する

1行目　変更内容
2行目　空行
3行目　変更理由

▼vimエディター操作方法
・insert(編集)モード:i
・内容を保温してVimエディターを閉じる:esc → :wq → enter

・git add
・git commit
・git rm
